Key Features

Love the boxed theme? Keep it. If not, try curving it!

Arc backend theme comes with two major themes within it- Curved & Box. By selecting one, the entire backend will adhere to the format and reflect the changes.

App Drawer

The app drawer section comprises all the applications that you have installed in your Odoo backend. This screen can only be found in the Enterprise version of Odoo. However, the Arc backend theme provides this section to all the users! There are two sub sections named- All & Frequently. It also has a global search button from where the user can search and navigate to any section he/she wishes to.

Mark Apps As Favorite &
Favorite Bar

Using the customize button in the top right corner of the app drawer section, one can mark their favorite apps from the app drawer. The favorite apps will then be listed in the Favorite Bar. It can be configured to be shown/hidden either on the left or bottom.

Color Theme

Change the color of your Odoo theme with predefined and custom themes. Using custom themes, the Odoo backend color combination(body, menu, tabs, etc) can be created from scratch while predefined themes give you on-the-go choices!

Mobile Responsive

Arc theme is now mobile responsive! The theme can now be accessed from the phone and the UI blends with the smartphone screens seamlessly.

Dark & Light Mode With Auto-detection

Arc theme has the trendy light & dark mode which the users can switch between. It also comes with a smart auto-detection feature that automatically switches on the dark/light mode based on the user preference.

Dark

Bookmark

Magnifier & Fullscreen mode

Using the magnifier button, you can magnify a specific portion of the screen. The Fullscreen mode lets you dive into the fullscreen of the Odoo backend

Q.1. What is the difference between full screen and magnifier?

Full screen mode is meant for switching the backend theme into the entire screen of the device that the user is using.
On the other hand, Magnifier can be used to highlight or zoom into a specific section of the screen that is visible. For example, if you want to focus on a particular area of the order list to look on some specific details, Magnifier can come handy.

Q.2. Does the admin can change any settings for the theme?

Yes, the admin can change any settings for the theme.

Q.3. What is the Odoo Arc Theme?

Odoo Arc theme is the best backend theme for Odoo that users can utilize and implement. It makes the otherwise boring backend of Odoo an exceptional one and the backend operators love it.

Q.4. What is the Favorites Icon and its Purpose?

Using the Favorite icon, one can save and access the favorite Odoo apps at one place.

Q.5. Can you customize the Menu Bar in Odoo Arc Theme?

Yes, you can definitely do some customizations from your end. You can place the Menu Bar horizontally and vertically. You can also add the company's logo. You can change the user profile’s position as well.

Q.6. What is the difference Between a Favorite Bar and Bookmark?

The Favorite Bar lists the applications that you have marked as favorite while Bookmark is a completely different thing. Using the Bookmark feature, you can save a specific page/segment inside an Odoo application. When you use the Bookmark, the current URL of the Odoo is saved and the user can create several bookmarks and name them.

Q.7. What Is the Purpose of Full Screen? How does that help?

There are many times when you need to focus on the screen by eliminating the unnecessary things. The full screen mode allows you to expand the Odoo screen to its fullest depending on the screen of the device.

Q.8. What Are the Benefits of a Magnifier?

Often, there are a lot of data sets that clog the screen and it is difficult for the human eye to properly comprehend them. The Magnifier helps to zoom into a specific section of the screen as per the requirement for better data visualization.
During an office presentation, the Magnifier can help by enlarging a segment and helping the audience see better.

Q.9. Is It Available in Multi-language?

Yes, it is available in multiple languages so that the theme can adapt to various countries.

Q.10. How to Add or Remove Apps in the Favorite Bar?

On clicking the Favorite button, all the available applications will be shown. They will have a small star marker placed on their head. If you select any app, the star marker will be filled and the app will be marked as the Favorite! If you deselect an already star-marked app, the app will be removed from the Favorite Bar.

Q.11. Is It Possible to Manage Two Types of Themes?

There are two major types of themes that the Arc theme is based on- curved/box. You can apply one of them as per requirement.

Q.12. Do You Provide Any Free Support?

Yes, just like any other Odoo app and theme developed by Ksolves, Arc Backend Theme is also backend and supported by the developers for 120 days(free of cost).

Q.13. Need Some Customization in This App, Whom to Contact?

You can feel free to contact the Ksolves team directly for customizations and modifications as per your requirement and we will definitely do it if it is a feasible proposition.
